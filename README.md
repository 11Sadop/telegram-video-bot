# 🎬 بوت الفيديوهات الشامل - تيليجرام

بوت تيليجرام متكامل لتحميل وتعديل الفيديوهات:
- **تحميل** من أكثر من 1000 منصة (تيك توك، يوتيوب، إنستجرام، X، فيسبوك)
- **معالجة** فيديوهاتك الشخصية المرفوعة من الجهاز
- **فلتر محتوى ذكي** لرفض المواد غير اللائقة

---

## ✨ الميزات الكاملة

### 📥 التحميل والمعالجة
| الميزة | الوصف |
|--------|--------|
| 📥 تحميل بدون علامة مائية | من أي منصة (تيك توك وغيرها) |
| 🎧 استخراج MP3 | تحويل الصوت لملف منفصل |
| 🔇 إزالة الموسيقى | إبقاء الكلام فقط |
| 🗣 إزالة الكلام | إبقاء الموسيقى فقط (جديد!) |
| 🔕 كتم الصوت | إزالة الصوت كلياً (جديد!) |
| 📺 رفع الجودة | إلى 1080p |
| ✂️ قص المقطع | تحديد بداية ونهاية |

### 🔄 طريقتان للاستخدام
1. **رابط:** أرسل رابط من أي منصة
2. **رفع مباشر (جديد!):** ارفع فيديو من جهازك مباشرة للبوت

### 🛡 فلتر محتوى متعدد الطبقات
- الطبقة 1: حجب المواقع الإباحية
- الطبقة 2: فحص عنوان ووصف الفيديو
- الطبقة 3: فحص إطارات الفيديو (NudeNet AI)
- الطبقة 4: فحص الكلام (Whisper)

## ⚡ تحسينات السرعة

> إذا كان البوت بطيء عندك، عنده حلول مدمجة:

### الخيار 1: سريع جداً (بدون فحص AI)
```bash
ENABLE_FRAME_CHECK=false
ENABLE_AUDIO_CHECK=false
```
- ✅ تحميل خلال **5-10 ثواني** بس
- ✅ لا يحتاج ذاكرة كبيرة (يشتغل على Render Free)
- ⚠️ الطبقتين 1 و 2 تبقى شغالة (حظر المواقع + فحص العنوان)

### الخيار 2: متوازن (موصى به)
```bash
ENABLE_FRAME_CHECK=true
ENABLE_AUDIO_CHECK=false
FRAMES_TO_CHECK=3
```
- ✅ فحص الصور فقط (~10 ثواني إضافية)
- ✅ يكشف العري في الفيديو
- ⚠️ يحتاج 700 MB RAM تقريباً

### الخيار 3: حماية كاملة (الأبطأ)
```bash
ENABLE_FRAME_CHECK=true
ENABLE_AUDIO_CHECK=true
```
- ✅ فحص صور + فحص كلام (يشتغلان **بالتوازي**)
- ⏱ ~20-30 ثانية إضافية
- ⚠️ يحتاج 1-1.5 GB RAM (يستلزم Render Starter $7/شهر)

## 💡 ليش كان البوت بطيء قبل؟

**المشكلة كانت:**
1. التحميل كان `bestvideo+bestaudio` ويحتاج دمج = بطيء
2. الفلاتر كانت تشتغل **واحدة ورا الثانية** = إضافة وقت
3. فحص 5 إطارات + 3 دقائق صوت = كثير

**الحلول المطبّقة:**
1. ✅ التحميل أصبح `best` (ملف واحد جاهز) = **أسرع ×3**
2. ✅ الفحص يشتغل **بالتوازي** (asyncio.gather) = **أسرع ×2**
3. ✅ فحص 3 إطارات بدل 5 + دقيقة صوت بدل 3 = **أسرع ×3**
4. ✅ `preset=veryfast` في ffmpeg بدل `fast` = **أسرع ×1.5**
5. ✅ `concurrent_fragment_downloads=4` في yt-dlp

**المحصلة:** البوت أسرع بـ 4-6 أضعاف تقريباً.

## 🚀 النشر على Render

### 1. توكن من [@BotFather](https://t.me/BotFather)

### 2. رفع الكود على GitHub
```bash
cd video_bot
git init
git add .
git commit -m "Initial"
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

### 3. Render → Background Worker
- **Build:** `apt-get update && apt-get install -y ffmpeg && pip install -r requirements.txt`
- **Start:** `python bot.py`
- **Environment Variables:**
  - `BOT_TOKEN` → التوكن
  - `ENABLE_FRAME_CHECK` → `false` (للخطة المجانية)
  - `ENABLE_AUDIO_CHECK` → `false` (للخطة المجانية)

## 📝 كيف تستخدم البوت؟

### تحميل من رابط:
1. انسخ رابط فيديو من أي منصة
2. ألصقه في البوت
3. انتظر الفحص والتحميل
4. اختر العملية من القائمة

### معالجة فيديو من جهازك:
1. افتح الفيديو في تيليجرام
2. اضغط "مشاركة" → اختر البوت
   - أو افتح البوت واضغط الملف المرفق وارفع الفيديو
3. اختر العملية مباشرة

> 💡 الفيديوهات المرفوعة من جهازك **ما تمر بفلتر المحتوى** (لأنها مقاطعك الشخصية).

## 🆘 حل المشاكل الشائعة

| المشكلة | الحل |
|---------|------|
| البوت بطيء جداً | `ENABLE_FRAME_CHECK=false` و `ENABLE_AUDIO_CHECK=false` |
| ينهار البوت (OOM) | نفس الحل السابق، أو ترقية للخطة المدفوعة |

| الفحص يرفض فيديو نظيف | غير `FRAMES_TO_CHECK=5` أو عدّل كلمات `content_filter.py` |
