# 🎬 بوت تحميل الفيديوهات مع فلترة المحتوى

بوت تيليجرام لتحميل الفيديوهات من أكثر من 1000 منصة (تيك توك، إنستجرام، يوتيوب، تويتر/X، فيسبوك، وغيرها) بدون علامة مائية، مع **نظام فلترة محتوى ذكي متعدد الطبقات** لرفض المحتوى غير اللائق.

## ✨ الميزات

### ميزات التحميل
- 📥 **تحميل بدون علامة مائية** من جميع المنصات الرئيسية
- 🎧 **استخراج الصوت** كملف MP3
- 🔇 **إزالة الموسيقى الخلفية**
- 📺 **رفع الجودة** إلى 1080p
- ✂️ **قص جزء معين** من المقطع

### نظام الحماية (4 طبقات)
- 🔒 **الطبقة 1:** حجب النطاقات الإباحية المعروفة (200+ موقع) فوراً
- 🔍 **الطبقة 2:** فحص عنوان، وصف، تصنيفات، وقناة الناشر
- 🖼 **الطبقة 3:** فحص إطارات الفيديو باستخدام NudeNet (AI)
- 🎤 **الطبقة 4:** تحويل الصوت لنص عبر Whisper وفحصه

يكفي أن تفشل أي طبقة ليتم رفض الفيديو تلقائياً 🚫

## 📋 المتطلبات

- Python 3.11+
- FFmpeg
- حساب تيليجرام + توكن من [@BotFather](https://t.me/BotFather)
- حساب Render.com (مجاني)

⚠️ **مهم جداً حول Render:** المكتبات الإضافية (nudenet + faster-whisper + onnxruntime) تحتاج **~1-1.5 GB RAM** وقت التشغيل. الخطة المجانية في Render تعطيك 512 MB فقط. خياراتك:

| الخيار | التكلفة | ملاحظات |
|--------|----------|----------|
| **Render Free** | مجاني | ⚠️ قد ينهار بسبب الذاكرة. يفضل تعطيل فحص الإطارات |
| **Render Starter** | $7/شهر | ✅ 512MB-2GB RAM حسب الاستخدام - موصى به |
| **VPS بديل** (Contabo/Hetzner) | $4-5/شهر | ✅ 4-8GB RAM - الأفضل للأداء |

## 🚀 خطوات النشر على Render

### 1. الحصول على توكن البوت
- افتح تيليجرام، ابحث عن [@BotFather](https://t.me/BotFather)
- أرسل `/newbot` واحفظ التوكن

### 2. رفع الكود على GitHub

```bash
cd video_bot
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

### 3. النشر على Render

1. [render.com](https://render.com) → سجل دخول بـ GitHub
2. **New +** → **Background Worker**
3. اربط المستودع
4. الإعدادات:

   | الحقل | القيمة |
   |-------|---------|
   | Name | `video-download-bot` |
   | Runtime | `Python 3` |
   | Build Command | `apt-get update && apt-get install -y ffmpeg && pip install -r requirements.txt` |
   | Start Command | `python bot.py` |
   | Plan | `Starter` (موصى به) |

5. **Environment Variables:**
   - `BOT_TOKEN` → التوكن من BotFather
   - `ENABLE_FRAME_CHECK` → `true` (أو `false` لتوفير الذاكرة)
   - `ENABLE_AUDIO_CHECK` → `true` (أو `false`)

6. **Create Background Worker**

## 🛡 تفاصيل نظام الفلترة

### الطبقة 1: فحص النطاق (فوري)
قائمة بأكثر من 200 موقع إباحي معروف. بمجرد وصول الرابط، البوت يرفضه دون أي معالجة. الملف: `content_filter.py`.

### الطبقة 2: البيانات الوصفية (سريع، < ثانية)
- فحص `age_limit` (إذا كان ≥ 18 يرفض فوراً)
- فحص العنوان، الوصف، التصنيفات، اسم القناة
- يستخدم قائمة كلمات مفتاحية عربية + إنجليزية

### الطبقة 3: فحص الإطارات (5-15 ثانية)
- يستخرج 5 إطارات من مواضع متفرقة بالفيديو
- يمررها على نموذج **NudeNet** (يكتشف العري بدقة عالية)
- التصنيفات المرفوضة: عضو ذكوري/أنثوي/ثدي/مؤخرة/فتحة شرج
- الملف: `nsfw_detector.py`

### الطبقة 4: فحص الكلام (15-60 ثانية)
- يحوّل الصوت إلى نص باستخدام **faster-whisper** (نموذج tiny)
- يدعم العربية والإنجليزية
- يفحص أول 3 دقائق فقط لتوفير الوقت
- الملف: `transcriber.py`

### تعطيل الفلاتر (عند الضرورة)
إذا الذاكرة قليلة، يمكن تعطيل الطبقات الثقيلة:

```bash
ENABLE_FRAME_CHECK=false  # يعطل فحص الإطارات (NudeNet)
ENABLE_AUDIO_CHECK=false  # يعطل فحص الكلام (Whisper)
```

**حتى لو عطلت الاثنين، الطبقتين 1 و 2 تبقى تعمل** (حجب النطاقات + فحص العنوان).

## ⚠️ ملاحظات

### الدقة
- **فحص النطاقات:** دقة 100% للمواقع المدرجة
- **NudeNet:** دقة ~90-95% (قد يعطي false positive على ملابس السباحة أو ملابس رياضية ضيقة)
- **Whisper tiny:** دقة ~85% للعربية الفصحى، أقل للعاميات

### الأداء
- فيديو 30 ثانية → فحص كامل ~20-30 ثانية
- فيديو 3 دقائق → فحص كامل ~45-60 ثانية
- يمكن تعطيل الفحص الثقيل عبر env vars

### التكلفة
- Render Free: صعب يتحمل كل الفلاتر مع الذاكرة 512MB
- Render Starter ($7/شهر): يشتغل بسلاسة
- VPS خارجي (Hetzner CX22 = $4/شهر): الأرخص والأقوى

## 🆘 حل المشاكل

| المشكلة | الحل |
|---------|------|
| البوت ينهار (Out of Memory) | ارفع الخطة لـ Starter أو عطل `ENABLE_FRAME_CHECK` |
| "nudenet not installed" | تأكد من `pip install -r requirements.txt` |
| أول طلب بطيء جداً | طبيعي - النماذج تحمّل أول مرة (~150MB) |
| فحص خاطئ (رفض فيديو سليم) | أضف كلمات/نطاقات لـ allowlist في `content_filter.py` |
